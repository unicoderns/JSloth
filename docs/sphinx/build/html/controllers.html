Controllers¶

The place for the business logic, there’s 4 categories of controllers:

JSloth Library¶

A reference to JSloth Library is available under this.jsloth

Local config¶

Each app can use a custom configuration, a reference of the object holding that data is available under this.config

Categories¶

  • RoutesController: Controller Hub, help you install other controllers as childs.
  • APIController: Parent Endpoint Controller
  • HtmlController: Parent Html-like Render Controller
